# Calculator_html_css_javascript

This project is a functional web-based calculator developed to demonstrate core competencies in front-end development. It focuses on the seamless integration of HTML5, CSS3, and JavaScript, specifically highlighting efficient DOM manipulation and event handling to create an interactive user interface.

##  What does it do?

Basically, exactly what you’d expect from a calculator:

 Standard Math: It adds, subtracts, multiplies, and divides.

 Split Screen: Instead of just one bar, we put 4 slots at the top so you can see the first number, the operator, the second number, and the result separately. This way, you don't get lost!

Deleting: You have the "C" button to clear everything and the backspace arrow to delete just the last digit if you make a mistake.

Error Handling: If you try to divide by zero, it shows an "Error" message so nothing breaks.

##  How we built it

1. The Skeleton (HTML): I used simple tags and buttons. The coolest part is the screen area, where I used read-only inputs to display the data.
2. The Look (CSS): I used Grid to layout the buttons quickly and keep everything aligned. The main color is a darkslategrey, with a pop of red on the equal button so it stands out. I also used the Poppins font because it looks way cleaner than the default ones.
3. The Magic (JavaScript): I used a for loop so all buttons work without having to code them one by one. The script checks if you clicked a number, a sign, or the equal button, and then does the math step-by-step.

##  How to run it

It's super simple:

1. Download the files.
2. Open index.html in your favorite browser.

That’s it! Start crunching some numbers.
